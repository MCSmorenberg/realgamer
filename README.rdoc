== README Rails API

== App installation


- bundle install
- rails server
- open http://localhost:3000


== Database installation


- rake db:create
- rake db:migrate
- rake db:seed


== Find data on the API

See the seeds file for the seeded data. For example you can see the data by putting this "../games/3/reviews" in the URL. It is about the games Id, "../games/ID/reviews".
